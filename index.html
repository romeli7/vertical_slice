Morocco's Phosphate Value Chain

  • Morocco holds 70% of global phosphate reserves.
  • Phosphates are vital for fertilizers, food, detergents, and industrial processes.
  • This map shows Morocco's extraction, processing, and export locations.
  • It reveals where value is created domestically versus captured abroad.
